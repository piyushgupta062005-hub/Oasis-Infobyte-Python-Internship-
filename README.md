# Oasis Infobyte - Python Programming Internship

This repository contains the projects completed during my 4-week Python Programming Internship at Oasis Infobyte. 

## Projects Included

### 1. Voice Assistant (Project 1)
- **Description:** A basic command-line voice assistant that responds to greetings, tells the current time/date, and performs quick Google searches.
- **File:** `project1.py`

### 2. BMI Calculator (Project 2)
- **Description:** A command-line tool that prompts users for their weight and height, calculates their Body Mass Index (BMI), and classifies their health category.
- **File:** `project2.py`

### 3. Random Password Generator (Project 3)
- **Description:** A secure password generator that creates strong, random passwords based on user-defined lengths using letters, digits, and special symbols.
- **File:** `project3.py`

## How to Run
1. Clone the repository or download the files.
2. Run any project using Python:
   ```bash
   python project1.py
   python project2.py
   python project3.py
